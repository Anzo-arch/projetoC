# 🚀 Projetos em Linguagem C – Disciplina ADS

Repositório com projetos práticos desenvolvidos durante o estudo de C, abordando:

- **Projeto 1:** Calculadora Modular (bibliotecas, E/S, arquivos)
- **Projeto 2:** Visualizador Gráfico de Dados (OpenGL/GLUT, eventos)
- **Projeto 3:** Estação Meteorológica Serial (portas de hardware, comunicação)
- **Projeto 4:** Servidor Multithread de Monitoramento (processos, threads, rede)
- **Projeto Integrador:** Dashboard IoT com Hardware Simulado (todos os conceitos)

Cada pasta contém código fonte, instruções de compilação e exemplos de uso.

## Como usar

Todos os projetos compilam com **GCC** e foram testados no **Ubuntu 22.04**.
